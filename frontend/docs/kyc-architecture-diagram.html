📋 KYC Lifecycle — Step-by-Step Flow

1 Customer Registration

Customer submits personal details (name, email, mobile) via mobile app or web portal. Session initialized.

2 Mobile Verification

OTP sent to registered mobile number. Customer verifies identity by entering OTP. SIM-based authentication.

3 Aadhaar Verification

Customer enters Aadhaar number. OTP sent to Aadhaar-linked mobile. UIDAI validates demographic data.

4 PAN Verification

PAN number validated for format & checksum. Cross-checked with Income Tax database for name match.

5 Face Verification

Live selfie captured with liveness detection (blink/movement check). Face matched against Aadhaar photo via AI.

6 Document Verification

Uploaded documents scanned via OCR engine. Data extracted, verified for authenticity and tampering.

7 Risk Assessment

All verification events consumed by Fraud Detection, AML, and Risk Assessment services for scoring.

8 Final Decision

If all checks pass → KYC Approved, account activated. If any check fails → KYC Rejected, customer notified with reason.
